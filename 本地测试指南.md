## 本地测试步骤

### 1. 启动开发环境

双击运行：
```
start-dev.cmd
```

这会启动：
- 后端服务：http://localhost:5678
- 前端服务：http://localhost:3012

### 2. 访问测试

打开浏览器访问：http://localhost:3012

### 3. 检查项目

- [ ] 页面底部是否显示贡献者信息
- [ ] "ComfyUI 贡献者: 你们喜爱的老王" 文字是否正确
- [ ] B 站按钮是否可点击
- [ ] 点击 B 站按钮是否跳转到 https://space.bilibili.com/97727630
- [ ] 版本号 v1.0.6-comfyui 是否显示
- [ ] 切换深色/浅色主题，footer 样式是否正常
- [ ] 缩小浏览器窗口，检查响应式布局是否正常

### 4. 停止服务

测试完成后，关闭两个命令行窗口即可停止服务

## 如果满意

测试通过后，可以：

1. 提交更改到 Git
2. 推送到 GitHub 仓库

命令：
```bash
cd 火宝一键漫剧/huobao-drama
git add web/src/components/common/AppLayout.vue
git commit -m "feat: 在 UI 底部添加 ComfyUI 贡献者信息"
git push comfyui master:main
```

## 注意事项

- 确保端口 5678 和 3012 未被占用
- 首次启动可能需要安装依赖（npm install）
- 如果前端启动失败，进入 web 目录手动运行 `npm install`
